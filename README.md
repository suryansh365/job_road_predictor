# 💼 Job Role Prediction System

A Machine Learning project that predicts the most suitable job role based on a student's technical skills.

---

## 🚀 Features

- Predicts suitable job roles
- User-friendly Streamlit interface
- Random Forest Machine Learning model
- Fast prediction
- Easy to deploy

---

## 🛠 Technologies Used

- Python
- Streamlit
- Scikit-Learn
- Pandas
- NumPy
- Matplotlib
- Joblib

---

## 📂 Project Structure

Job-Role-Prediction/
│
├── app.py

├── requirements.txt
├── README.md
└── dataset.csv

---

## 📊 Machine Learning Workflow

1. Load Dataset
2. Data Preprocessing
3. Encode Labels
4. Split Training and Testing Data
5. Train Random Forest Classifier
6. Evaluate Model
7. Save Model
8. Deploy using Streamlit

---

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/job-role-prediction.git
```

Move into the project folder

```bash
cd job-role-prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
